aeon.io.video¶

aeon.io.video.export(frames, file, fps, fourcc=None)[source]¶

Exports the specified frame sequence to a new video file.

Parameters:

  • frames (iterable) – An object to iterate over the raw video frame data.

  • file (str) – The path to the exported video file.

  • fps – The frame rate of the exported video.

  • fourcc (optional) –

Specifies the four character code of the codec used to compress the frames.

aeon.io.video.frames(data)[source]¶

Extracts the raw frames corresponding to the provided video metadata.

Parameters:

data (DataFrame) –

A pandas DataFrame where each row specifies video acquisition path and frame number. :return: An object to iterate over numpy arrays for each row in the DataFrame, containing the raw video frame data.
